# Women Safety Analysis Using Machine Learning

## Project Overview
This project analyzes women safety using machine learning, sentiment analysis, and crime statistics data.

## Datasets Used
1. Sentiment140 Dataset
2. Crimes Against Women Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- NLTK
- Scikit-learn
- Matplotlib
- Seaborn

## Machine Learning Model
- TF-IDF Vectorization
- Logistic Regression

## Results
- Accuracy: 76.1%
- Sentiment Classification
- Crime Trend Analysis
- State-wise Crime Analysis
- Word Cloud Visualization

## Features
- Tweet preprocessing
- Sentiment prediction
- Crime analysis
- Data visualization
